# Clustered-JM-Dynamic-Prediction
R code for the paper "Joint Models of Two Longitudinal Biomarkers and Clustered Survival Data for Dynamic Prediction with Application to Periodontitis".

## sample_survdata.csv 
Example data set for the survival submodel

## sample_longdata.csv
Example dataset for the longitudinal submodel 

## dynamic prediction function.R
Function to perform dynamic prediction of survival outcome of a selected unit from a cluster 

## sample code.R
R code to apply the proposed joint model for clustered data and to perform dynamic prediciton 

## clustered_JM_2biom_correlated_application.stan
The stan code for fitting the joint model
