# Clustered-JM-Dynamic-Prediction
R code for the paper "Joint Models of Two Longitudinal Biomarkers and Clustered Survival Data for Cluster-Informed Dynamic Prediction with Application to Periodontitis".

## sample_survdata.csv 
Example data set for the survival submodel

## sample_longdata.csv
Example dataset for the longitudinal submodel 

## dynamic prediction function.R
Function to perform dynamic prediction

## sample code.R
R code to apply the proposed joint model for clustered data

## clustered_JM_2biom_correlated_application_lkj_weibull.stan
The stan code for fitting the joint model  

## analysis_DP_allinfo.stan
The stan code to perform cluster-informed dynamic prediction
